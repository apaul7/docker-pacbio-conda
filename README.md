# docker-pacbio-conda
docker image for running analysis on PacBio data.

So far only supports python2

Currently contains:
- pbsv version 2.3.0
- pbccs version 4.2.0
- pbbam version 1.0.7
- pbmm2 version 1.2.1
- lima verion 1.11.0

other potential conda packages can be found https://github.com/PacificBiosciences/pbbioconda
